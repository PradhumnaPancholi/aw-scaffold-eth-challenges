# Start
Terminal 1: `yarn chain`
Terminal 2: `yarn start`
Terminal 3: 
```
yarn run generate
yarn account
// add faucet
yarn deploy --reset
yarn build
yarn surge
yarn test
yarn verify --network mainnet #here need to use mainnet for API is in mainnet even though I am running on ropsten testtnet
```
